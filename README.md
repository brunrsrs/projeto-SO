Pra compilar usar o comando:

gcc main.c functions.c -lpthread -o main -lm

O "functions.c" é adicionado porque ele é usado na main também e, diferentemente da .h, ela não é uma biblioteca e sim um arquivo de funções (daí precisa ser compilada separadamente também)
A math.h é utilizada para a operação ceiling e precisa ser adicionada na compilação

O programa funciona em 3 threads + a main:
- A main é dedicada para operações interativas como menu e apresentação de dados. 
- A função "exec" é a principal que fará a leitura do arquivo inserido e as suas devidas operações.
- As demais 2 funções são auxiliares para funções de entrada/saida e para lista de espera para semáforos
    - Ambas essas 2 funções elas basicamente colocam o programa em espera até cumprirem uma certa condição e depois retornam para a função principal (exec)

Alguns problemas encontrados:
- Após passar o programa para uma função auxiliar houveram problemas para leva-lá de volta à função principal podendo gerar problemas com a fila.
